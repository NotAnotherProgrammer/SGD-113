SGD 113 – C# Programming Exercises
Overview

This repository contains C# scripts and Unity-based exercises completed as part of SGD 113 (Programming for Simulation & Game Development).

The work in this repository focuses on building a strong foundation in programming concepts and applying them within a game development environment.
